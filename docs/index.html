What is Vyne?

Vyne is a systems programming language that aims to be an evolution of stack-based interpretation, adding modern quality-of-life features while maintaining a minimalistic ethos.

⚠️ Warning: Docs may not reflect current language state. The Vyne VM and compiler are still evolving.

Design Goals

  • The VM always does the pop: Predictable stack management.
  • Minimalistic: No feature should be unnecessary or redundant.
  • Tree-based Scoping: Using "Groups" to prevent global namespace pollution.

Hello World

A simple program in Vyne looks like this:

function main() {
    log("Hello, Vyne!");
}

Postfix Operators

Vyne supports ++ and --. These return the original value before the modification. 

var x = 10;
log(x++); // Outputs 10
log(x);   // Outputs 11
